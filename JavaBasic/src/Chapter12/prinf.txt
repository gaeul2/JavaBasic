// prinf() - 문자열의 조합 //
    서식지정자 수와 전달하는 인자의 수가 같아야 함
    == 서식 지정자 ==
        - %d : 10진수 정수 형태의 출력
        - %o : 8진수 정수 형태의 출력
        - %x : 16진수 정수 형태의 출력
        - %f : 실수의 출력
        - %e : e표기법 기반의 실수 출력
        - %g : 출력의 대상에 따라서 %e 또는 %f 형태의 출력
        - %s : 문자열 출력
        - %c : 문자 출력
